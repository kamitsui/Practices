# Project
In this project I installed minilibx and made some sample code.

## How to install minilibx
case: macOS
`minilibx_mms_20191025_beta.tgz`

For example, unzip the library in the following directory
`mkdir $HOME/lib/minilibx & cd $HOME/lib/minilibx`
`tar -xvf minilibx_mms_20191025_beta.tgz`
`$HOME/lib/minilibx/minilibx_mms_20191025_beta`
`cd $HOME/lib/minilibx/minilibx_mms_20191025_beta`
`make`
`cp libmlx.dylb $HOME/lib/libmlx.dylb`
